# Agentivum

Landing page for [Agentivum s.r.o.](https://agentivum.com) — AI engineering consultancy.

## Deployment

Hosted via GitHub Pages. Push to `main` to deploy.

### Custom domain setup

1. Set DNS A records for `agentivum.com` to GitHub Pages IPs:
   ```
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153
   ```
2. Set CNAME record: `www` → `zahorack.github.io`
3. Enable HTTPS in repo Settings → Pages
